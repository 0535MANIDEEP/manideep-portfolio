# Portfolio — Manideep Daram

Personal portfolio website showcasing projects, skills, experience, and achievements.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | React 19, TypeScript 6 |
| Build Tool | Vite 8 |
| Styling | Tailwind CSS 4 |
| State Management | Redux Toolkit, Redux Persist |
| Routing | React Router v7 |
| Animations | GSAP, Framer Motion |
| Forms | React Hook Form, Zod |
| Charts | Chart.js, React Chartjs 2 |
| HTTP Client | Axios |
| Email | EmailJS |
| Carousel | Swiper |

## Features

- Single-page application with client-side routing
- Responsive design with smooth scroll animations (Lenis)
- Dark/light theme support
- Contact form with email integration
- Project showcase with detailed views
- Skills visualization with animated charts
- Testimonials carousel
- PDF resume download
- SEO optimization via React Helmet

## Setup

```bash
npm install
npm run dev
```

The dev server runs at `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```
